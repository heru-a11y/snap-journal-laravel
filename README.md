## ğŸ§  Overview
**Snap Journal** is an AI-based visual journaling application that helps users reflect on their daily activities through notes, photos, and automatic insights generated by AI.  
The application is built with **Laravel 12** as the backend, **React.js (Inertia)** for the frontend, and **FastAPI + LangChain** as the AI integration layer.
